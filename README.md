# Software para Desktop de Login e SignUp

Este projeto consiste no desenvolvimento de um sistema de Login e Cadastro em Python, no qual o usuário pode criar uma conta e, posteriormente, autenticar-se para verificar se ela foi corretamente registrada no sistema.

Para a implementação, foram utilizadas as bibliotecas PySide6 e SQLite. A PySide6 foi responsável pela construção da interface gráfica do software, possibilitando a interação com o usuário, enquanto o SQLite foi empregado como banco de dados relacional para armazenar e gerenciar as informações das contas criadas.

 # Objetivo desse projeto

O principal objetivo no desenvolvimento deste software é demonstrar minhas competências em programação com Python, por meio da criação de aplicações desktop que valorizam tanto a identidade visual quanto a estrutura de backend, com ênfase na modelagem e implementação de bancos de dados SQL.
